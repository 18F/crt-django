Methodology

Document count is the number of complaints that the word appears in. Word count is the total number of times that word appears.

The data is based on creation dates. Time periods start on Sunday, except for year. The cutoff is midnight GMT.

There my be variations in the document count and search results. For the search, results will be shown for variations on a word, so "pet" and "pets" will show up on the view all table if you search for "pet" or "pets" in the description. The counts on this page are using exact word matches, so "pet" and "pets" will be counted separately. Capitalization is not a factor when searching or counting so searching "Pet" will not be different than "pet". The counts for this page are also optimized for English characters.
